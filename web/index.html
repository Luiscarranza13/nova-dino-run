CARACTERÍSTICAS

Todo lo que hace a Nova Dino Run único

🦕

Dino Animado

Sprites propios con 4 frames de carrera, animación de salto, agachado y muerte con efectos de partículas.

🌵

3 Obstáculos

Cactus, pájaros voladores y rocas generados aleatoriamente. La dificultad aumenta con el tiempo.

⚡

Power-Ups

Escudo (absorbe un golpe), Doble Salto y Turbo con barras de duración visibles en pantalla.

🎵

Música Chiptune

Banda sonora retro generada proceduralmente estilo Pou. Efectos de sonido para cada acción.

🌃

Ciudad Nocturna

Fondo parallax con silueta de ciudad, luna, estrellas parpadeantes y nubes animadas en 3 capas.

🏆

Hi-Score

Tu mejor puntaje se guarda entre sesiones. Flash rojo al morir y animación de nuevo récord.

COMPILAR DESDE FUENTE

Requiere CMake ≥ 3.16 y MinGW-w64

# Clonar
git clone https://github.com/Luiscarranza13/nova-dino-run.git
cd nova-dino-run

# Configurar (descarga Raylib automáticamente)
cmake -B build -DCMAKE_BUILD_TYPE=Release -DCMAKE_POLICY_VERSION_MINIMUM=3.5

# Compilar y ejecutar
cmake --build build --config Release
./build/NovaDinoRun.exe
